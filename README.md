# 🍕 Way Fome - Aplicativo de Delivery

Um aplicativo moderno e completo para serviço de delivery de comida, conectando clientes, restaurantes e entregadores.

## 📋 Características Principais

- ✅ Autenticação de usuários (clientes, restaurantes, entregadores)
- ✅ Catálogo de restaurantes e produtos
- ✅ Carrinho de compras e checkout
- ✅ Rastreamento de pedidos em tempo real
- ✅ Sistema de avaliações e comentários
- ✅ Pagamento integrado
- ✅ Dashboard para restaurantes
- ✅ Mapa interativo para entregadores

## 🏗️ Arquitetura do Projeto

```
way-fome/
├── frontend/          # Aplicação React/Web
├── mobile/            # Aplicativo React Native
├── backend/           # API Node.js/Express
├── docs/              # Documentação
└── README.md
```

## 🔧 Stack Tecnológico

### Frontend
- **React** - Interface web
- **Vite** - Build tool
- **Tailwind CSS** - Estilização
- **Redux** - State management

### Mobile
- **React Native** - App mobile
- **Expo** - Desenvolvimento

### Backend
- **Node.js** - Runtime
- **Express** - Framework
- **PostgreSQL** - Banco de dados
- **JWT** - Autenticação

## 🚀 Como Começar

```bash
# Clone o repositório
git clone https://github.com/leilaluanale13-jpg/Way-fome-.git
cd Way-fome-

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

## 📱 Usuários do Sistema

1. **Cliente** - Faz pedidos
2. **Restaurante** - Gerencia pedidos e cardápio
3. **Entregador** - Realiza entregas
4. **Admin** - Gerencia o sistema
